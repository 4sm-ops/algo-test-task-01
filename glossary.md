# Глоссарий терминов алготрейдинга

## Биржи

| Термин | Расшифровка | Описание |
|--------|-------------|----------|
| **B3** | Brasil, Bolsa, Balcão | Бразильская фондовая биржа, крупнейшая в Латинской Америке |
| **MOEX** | Moscow Exchange | Московская биржа, основная торговая площадка России |

## Финансовые инструменты

| Термин | Описание |
|--------|----------|
| **Фьючерс** | Производный финансовый инструмент — контракт на покупку/продажу базового актива в будущем по заранее оговорённой цене |
| **WDO** | Мини-контракты на курс доллара США к бразильскому реалу на бирже B3. Каждый контракт = $10,000 |
| **GLDG26** | Фьючерс на золото на бирже B3 с экспирацией в феврале 2026 (G = февраль) |
| **GOLD-3.26** | Фьючерс на золото на MOEX с экспирацией в марте 2026 |

## Торговые стратегии и понятия

| Термин | Описание |
|--------|----------|
| **Арбитраж** | Стратегия извлечения прибыли из разницы цен одного актива на разных рынках или связанных активов. Покупка дешевле, продажа дороже одновременно |
| **Календарный спред** | Разница цен между фьючерсами на один базовый актив с разными сроками экспирации. Используется для торговли временной структурой |
| **Волатильность** | Статистическая мера разброса доходности актива. Высокая волатильность = большие колебания цены |
| **Моментум** | Скорость и направление изменения цены. Стратегии моментума предполагают, что тренд продолжится |
| **Экспирация** | Дата истечения срока действия фьючерсного контракта |

## Рыночные данные

| Термин | Описание |
|--------|----------|
| **Маркетдата (Market Data)** | Поток рыночных данных: цены, объёмы, заявки в реальном времени |
| **Книга заявок (Order Book)** | Список всех активных заявок на покупку и продажу по инструменту |
| **Bid** | Лучшая цена покупки (заявка на покупку) |
| **Ask (Offer)** | Лучшая цена продажи (заявка на продажу) |
| **Spread (Спред)** | Разница между лучшей ценой продажи (ask) и лучшей ценой покупки (bid) |
| **Snapshot** | Полный снимок состояния книги заявок на определённый момент времени |
| **Incremental Updates** | Инкрементальные обновления книги заявок (добавление, изменение, удаление заявок) |

## Технические термины

| Термин | Расшифровка | Описание |
|--------|-------------|----------|
| **PCAP** | Packet Capture | Формат файлов для записи сетевого трафика. Используется для захвата и анализа биржевых данных |
| **Latency (Латентность)** | — | Задержка между событием и реакцией системы. В контексте задачи: 400 мс от получения данных до выставления заявки |

## Поля данных в CSV

| Поле | Описание |
|------|----------|
| **ts** | Timestamp — временная метка в наносекундах |
| **symbol** | Тикер (код) инструмента |
| **bid_price** | Лучшая цена покупки |
| **bid_qty** | Объём по лучшей цене покупки |
| **ask_price** | Лучшая цена продажи |
| **ask_qty** | Объём по лучшей цене продажи |
